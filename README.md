# Projeto-Multidisciplinar-Foco-em-qualidade-de-Software-
Projeto acadêmico multidisciplinar com foco em qualidade de software, aplicando na prática conhecimentos do curso de Análise e Desenvolvimento de Sistemas. Incluiu análise de requisitos, diagramas de caso de uso, requisitos funcionais e não funcionais, modelagem de processos, pseudocódigo e criação de casos de teste.
